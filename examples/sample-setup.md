# Example: what a generated setup looks like

*Illustrative only — values are made up to show the format, not a recommended setup.*

When you ask the tool to build a setup, it creates a row in your Notion `Setups` database and
writes a **phone-readable page** so you can read it on your phone while entering values in-game.
The page body looks like this:

---

## 🏁 Enter in-game — Lancia Stratos HF · "alsace gpt1" · Tarmac

**Tyres**
- Tyre type → Tarmac Soft

**Gearbox**
- Gear set → 2
- Primary gear → 33/31 × 31/30

**Suspensions — Front**
- Adjuster ring (ride height) → 0.043 m
- Spring stiffness → 65400 N/m

**Suspensions — Rear**
- Adjuster ring (ride height) → 0.082 m
- Spring stiffness → 32660 N/m

**Dampers — Front**
- Slow bump → 5000 · Slow rebound → 8000 · Fast bump → 4500 · Fast rebound → 7000

**Dampers — Rear**
- Slow bump → 3500 · Slow rebound → 6000 · Fast bump → 3000 · Fast rebound → 5000

**Axles**
- Front ARB → 2750 N/m · Rear ARB → 4250 N/m

**Differential — Rear**
- Ratio → 65/19 · LSD power/coast ramp → 60/70 · Preload → 70 Nm · Plates → 4

**Wheels**
- Pressure F/R → 32 / 32 psi · Camber F/R → −1.2° / −1.5° · Toe → front out / rear in

**Brakes**
- Front bias → 0.55

---

> ▸ **Why these values** *(toggle)*
> *Bumpy, fast tarmac → moderately stiff springs and a forward-ish brake bias for stability
> under hard braking; rear ARB stiffer than front for gentle throttle-on rotation `[RWD]`;
> LSD coast 70 keeps entry calm. (Cites your guideline: "I like the rear to rotate gently on
> throttle.")*

After driving it, set a **Rating**, add any **Notes**, and tick **Learn from this** if you like
it — future setups in `learn` mode will lean on the ones you've checked.
